# ProjectTR

## 使用機器

- Raspberry Pi 2
- DRV8835 (DUAL LOW VOLTAGE H-BRIDGE IC)
- LISD3H (MEMS digital output motion sensor)
- HMC5883L (3-Axis Digital Compass IC)
